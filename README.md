# 🎓 CampusConnect

CampusConnect is a Student Complaint Management System developed using Java, JSP, Servlets, JDBC, MySQL, Maven, Apache Tomcat, and Bootstrap.

## 🚀 Features

- Student Registration
- Student Login
- Admin Login
- Raise Complaint
- View My Complaints
- Admin Dashboard
- View All Complaints
- Resolve Complaints
- Session Management
- MySQL Database Integration

## 🛠️ Technologies Used

- Java
- JSP
- Servlets
- JDBC
- MySQL
- Apache Tomcat
- Maven
- Bootstrap 5

## 🗄️ Database

Database Name:

```
campusconnect
```

Tables:

- users
- complaints

## ▶️ How to Run

1. Clone the repository.
2. Import into Eclipse/VS Code.
3. Configure MySQL.
4. Build using Maven.
5. Deploy on Apache Tomcat.
6. Open:

```
http://localhost:8080/CampusConnect/
```
